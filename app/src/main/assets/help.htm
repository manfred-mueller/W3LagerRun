W3Web ist eine Anwendung, um W3Lager im Google Chrome oder Mozilla Firefox Browser zu starten. Der Grund hierfür liegt in der Optimierung der Scannertasten- und Kamera-Zugriffe in W3Lager.

  • Installationsprüfung
    W3Web überprüft, ob der Google Chrome Browser installiert ist.

  • Versionssprüfung
    Falls bereits installiert, überprüft W3Web, ob der Google Chrome Browser in einer Version größer 67 installiert ist.

  • W3Lager-URL
    Die URL zur Kunden-spezifischen W3Lager-Installation lässt sich in den Einstellungen frei festlegen. Die konfigurierte URL wird auf Plausibilität überprüft.

  • Aufruf W3Lager
    Die konfigurierte URL wird beim Start von W3Web automatisch aufgerufen. Wurde der Browser geschlossen, kann W3Lager durch Tippen auf das W3Coach-Logo in der Mitte des Bildschirms manuell wieder aufgerufen werden.

  • Einstellungen
    Neben Datum-/Uhrzeit- und WLAN- sind auch die Scanner-Einstellungen integriert.

    Optional kann in den Einstellungen der automatische Start von W3Web nach Systemstart konfiguriert werden.

    Falls beide installiert sind, kann in den Einstellungen zwischen Google Chrome und Mozilla Firefox umgeschaltet werden.

  • Integrationen
    Sofern installiert, werden Teamviewer® Quicksupport™, Chainway® Keyboard-Emulator und ein Keycode-Testtool eingebunden und können aus W3Web heraus aufgerufen werden.
